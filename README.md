
# CRUD Swagger API

This is a Spring Boot project with full CRUD support and an interactive Swagger UI.

## How to Run

1. Build and start:
```bash
mvn spring-boot:run
```

2. Access Swagger:
```
http://localhost:8080/swagger-ui.html
```

3. Use H2 console:
```
http://localhost:8080/h2-console
```

## Tech Stack
- Spring Boot
- Spring Data JPA
- H2 In-Memory DB
- Swagger UI (OpenAPI)
